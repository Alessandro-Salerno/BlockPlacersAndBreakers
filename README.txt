This branch contains a version of the BlockPlacersAndBreakers Server-Side Plugin compatible with Minecraft version 1.21.1.
This contribution was submitted by Alessandro Salerno (https://github.com/Alessandro-Salerno) on Sept 12th, 2024. 

Features added:
+ Works on 1.21.1 and newer versions in general (probably)

Features removed:
- WorldGuard support
- RedProtect support
- Economy support
- Pre-1.13 support

NOTE: Use this version on small servers EXCLUSIVELY. The only anti-grief system still included is GriefProtection, all others have been removed due to compilation issues. I am not liable for damages you may incur on your server because of this version of the Plugin.
